# eris chains clean

Clean a blockchains' data from the node.

## Synopsis

Clean a blockchains' data from the node.

Clean will remove the blockchain reference as well as
its data.

```bash
eris chains clean [name]
```

## Options

```
  -h, --help=false: help for clean
```
