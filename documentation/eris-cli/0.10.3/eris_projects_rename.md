# eris projects rename

Rename a project registered with Eris.

## Synopsis

Rename a project registered with Eris. To add a project use:
eris project add [project-definition-file]

```bash
eris projects rename [old] [new]
```

## Options

```
  -h, --help=false: help for rename
```
