# eris services ls

List the installed and built services.

## Synopsis

Lists the installed and built services which Eris knows about.

To list the known services: [eris services known]
To list the running services: [eris services ps]
To start a service use: [eris services start serviceName].

```bash
eris services ls
```

## Options

```
  -q, --quiet[=false]: machine parsable output
```
