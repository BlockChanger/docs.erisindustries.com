# eris chains add

Adds an existing blockchain to Eris' blockchain tree.

## Synopsis

Adds an existing blockchain to Eris' blockchain tree.

```bash
eris chains add [name] [ref]
```

## Options

```
  -h, --help=false: help for add
```
