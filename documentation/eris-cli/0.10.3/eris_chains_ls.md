# eris chains ls

Lists all known blockchains in the Eris tree.

## Synopsis

Lists all known blockchains in the Eris tree.

To list the known chains: [eris chains known]
To list the running chains: [eris chains ps]
To start a chain use: [eris chains start chainName].


```bash
eris chains ls
```
